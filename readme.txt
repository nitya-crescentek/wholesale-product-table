== Description ==

Wholesale Prices and Product Table displays a comprehensive wholesale product table on your site using a simple shortcode. It comes with various customizable options such as configurable columns, AJAX add-to-cart functionality, and variable product dropdowns that update product prices and images dynamically. The plugin also includes features like pagination, search, category filtering, and the ability to set global discounts based on user roles.

Features:
- Displays a customizable product table via a shortcode.
- Configurable columns for product information.
- AJAX-powered add-to-cart functionality for seamless shopping.
- Variable product dropdowns with live price and image updates.
- Pagination for large product catalogs.
- Product search functionality for quick filtering.
- Category-based product filtering for easy navigation.
- Set global discount for user roles (e.g., wholesale buyers).
- Full WooCommerce integration.

== Installation ==

1. Upload the `wholesale-prices-and-product-table` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided shortcode `[wholesale_product_table]` in any page or post where you want to display the table.

== Frequently Asked Questions ==

= How do I add the product table to my site? =

Simply use the shortcode `[wholesale_product_table]` on any page or post.

= Can I customize the columns shown in the product table? =

Yes, the plugin allows you to customize which columns are displayed in the product table via the plugin settings page.

= Does this plugin support variable products? =

Yes, the plugin supports WooCommerce variable products. When a user selects a variation, the product's price and image are updated automatically.

= Can I set different discounts for different user roles? =

Yes, you can configure global discounts for specific user roles (e.g., wholesale buyers) in the plugin settings.

= Does the plugin work with all themes? =

The plugin is designed to work with most themes, but there may be minor styling adjustments needed based on your theme.
